## Приложение для поиска и просмотра фильмов
Данное приложение было разработано в рамках учебной практики. Программа позволяет пользователям выполнять поиск фильмов через ПоискКино API, просматривать подробную информацию о картинах (рейтинг, жанры, описание, дата выхода), добавлять фильмы в избранное и список «смотреть позже», выставлять персональные оценки, а также просматривать статистику своей активности.

## Установка
Для установки программы необходимо:
1. Перейти в раздел Releases
2. Скачать файл `MovieApp.exe` из самого нового релиза
3. Запустить скачанный файл
4. При появлении предупреждения системы безопасности нажать «Подробнее» и «Выполнить в любом случае»

Либо собрать проект самостоятельно:
1. Клонировать репозиторий
2. Открыть `MovieApp.sln` в Visual Studio 2026
3. Выбрать Build → Build Solution
4. Запустить из папки `bin/Debug/MovieApp.exe`

## Требования к устройству:
- Windows 10 или выше
- .NET Framework 4.8
- Доступ к интернету для поиска фильмов через API Кинопоиска
- Минимум 4 ГБ ОЗУ
- 200 МБ свободного места на диске

## Использование
- **Регистрация** — создайте аккаунт с именем пользователя, email и паролем
- **Авторизация** — войдите в систему под своей учётной записью
- **Поиск фильмов** — введите название в строку поиска и получите карточки фильмов
- **Просмотр карточки** — нажмите «Подробнее» для получения полной информации о фильме
- **Избранное** — добавляйте фильмы в избранное для быстрого доступа
- **Смотреть позже** — сохраняйте фильмы в список запланированных к просмотру
- **Оценки** — выставляйте персональные оценки от 1 до 10
- **Статистика** — просматривайте свою активность: количество фильмов, среднюю оценку, графики
- **Просмотр онлайн** — переходите на страницу фильма на Кинопоиске для просмотра

## Технологии
- Язык: C#
- Интерфейс: Windows Forms
- Платформа: .NET Framework 4.8
- API: ПоискКино API
- Newtonsoft.Json
- Хранение данных: JSON (users.json, user_lists.json)
- Логирование: Текстовые файлы в папке Logs
